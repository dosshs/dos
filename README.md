## Installation

Use the package manager [npm](https://www.npmjs.com/) to install the dependencies.

frontend

```bash
dos\frontend> npm install
```

backend

```bash
dos\backend> npm install
```

## Usage

frontend

```javascript
dos\frontend> npm run dev
```

backend

```javascript
dos\frontend> npm start
```
